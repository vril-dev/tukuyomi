# tukuyomi-edge

`tukuyomi-edge` is the edge/device-oriented runtime in the Tukuyomi family.

The current public `tukuyomi` repository focuses on the WAF / reverse proxy /
API gateway product. Edge integration is planned separately; archived edge
release assets remain in `tukuyomi-releases`.

See [Product Comparison](../../product-comparison.md) for the current product
boundary.
